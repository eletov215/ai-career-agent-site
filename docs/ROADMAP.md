## Этап 8. MVP 1.0

MVP считается готовым, когда:

- пользователь может создать профиль;
- загрузить резюме;
- выполнить поиск;
- получить вакансии минимум из двух источников;
- сохранить вакансию;
- получить AI-анализ;
- подготовить текст ответа;
- самостоятельно перейти к отклику;
- все ключевые функции работают стабильно.
