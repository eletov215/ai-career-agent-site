Политика конфиденциальности

Последнее обновление: 16 июля 2026 года.

1. Какие данные могут обрабатываться

В тестовой версии сервис может обрабатывать контактные данные, настройки поиска, загруженное пользователем резюме и ссылки на вакансии.

2. Для чего используются данные

Данные используются для анализа резюме, сопоставления с вакансиями, подготовки рекомендаций и сопроводительных писем.

3. Доступ к сторонним сервисам

Сервис не запрашивает и не хранит пароли пользователей от сторонних площадок. Подключение внешних сервисов выполняется только через официальные механизмы авторизации, если они доступны.

4. Передача данных

Данные не продаются третьим лицам. Передача возможна только поставщикам инфраструктуры, необходимым для работы сервиса, либо когда этого требует закон.

5. Удаление данных

Пользователь сможет запросить удаление профиля и связанных данных. До публичного запуска контакт для таких запросов будет указан отдельно.

6. Тестовый статус

Сервис находится на стадии MVP. Политика будет уточнена до публичного запуска и начала обработки данных реальных пользователей.
